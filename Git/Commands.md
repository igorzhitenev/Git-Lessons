# Команды

Создание репозитория
```sh 
git init
```
Сохранение
```sh 
git add
```
Коммент к сохранению
```sh 
git commit -m "Text"
```
Смотреть изменения
```sh 
git diff
```

Журнал действий
```sh
git log
git log --oneline
git log --one line --graph
```
Вытащить сохранение
```sh
git checkout
git checkout master
git checkout head
```

Работа с ветками
```sh
git branch
git branch "new"
git merge "new"

 Конфликт при слиянии
выйти из доп.окна кнопкой q и далее
:wq

Удаление ветки
git branch -d "New"
```
